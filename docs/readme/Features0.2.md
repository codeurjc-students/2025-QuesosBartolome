## Features – Version 0.2

Below we present a detailed overview of the features included in version 0.2 of the website:

![Main Screen](/Images/MainPage.png)

- On this screen, a new feature has been added that allows users to filter the list of cheeses by type using the selector located at the top left.

![Cheese Page](/Images/CheesePageNR.png)

- On this page, we have added the ability to view the reviews of each cheese.

![Profile](/Images/ProfilePage.png)

- From the profile page, a user can manage their reviews, being able to view or delete them.

![Edit Profile](/Images/EditProfile.png)

- A registered user can also edit their profile as they wish, including both their credentials and other attributes such as name, profile picture, or address.

![Create Cheese](/Images/CreateCheese.png)

- A new feature has been added for administrators, allowing them to create new cheeses for the cheese factory.

![Make a Review](/Images/MakeReview.png)

- Registered users can rate a cheese by writing a review directly from its detail page.

![Clients](/Images/ClientsPage.png)

- From this screen, in addition to viewing clients as before, an administrator can ban users when deemed necessary.

![Stock](/Images/StockPage.png)

- From this screen, an administrator can manage the stock of all cheeses on the website, being able to add new boxes or remove existing ones.

![PreviewOrder](/Images/PreviewOrder.png)

- An administrator can process orders made by users, either confirming them and generating the invoice or rejecting them, in which case the order will be marked as denied.

![Invoices](/Images/InvoicesPage.png)

- From this screen, an administrator can view the different invoices for each order and download them in PDF format.
