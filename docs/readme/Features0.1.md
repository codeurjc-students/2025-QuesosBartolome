## Features – Version 0.1

Below we present a detailed overview of the features included in version 0.1 of the website:

![Main Screen](/Images/MainPage.png)

- On this screen you can see all the cheeses available on the website, along with basic details such as name and price per box. By selecting a cheese, you can access its detailed page. You can also see the buttons to register or log in, as well as the "About Us" section button on the left side.

![Cheese Page](/Images/CheesePage.png)

- This screen shows the detailed information of each cheese. If you are a registered user, you can place orders by boxes. If you are an administrator, an “Edit” button will appear, although it is not functional yet.

![LogIn](/Images/LoginPage.png)

- Form used to enter the credentials of a registered user or an administrator. The form validates the input data, and if they are incorrect, an error message is displayed as an alert.

![Register](/Images/RegisterPage.png)

- Form used to register a new user. The system validates the input data, and if they are not valid, the user is notified through an alert.

![About us](/Images/AboutUsPage.png)

- A page with minimal functionality that displays information about the cheese factory and its location using a Google Maps embed.

![Profile](/Images/ProfilePage.png)

- Screen that displays the user’s profile and personal information. Editing functionalities are not available yet.

![My order](/Images/MyOrderPage.png)

- “My Order” screen for registered users. It shows a cart with the cheeses the user has added. Items can be removed from the cart, and the order can be placed from this screen.

![Orders](/Images/OrdersPage.png)

- From this screen, administrators can view all orders placed to the cheese factory. Orders are displayed in pages of 10 items.

![Clients](/Images/ClientsPage.png)

- On this screen, administrators can view all clients (registered users) of the website. The list is paginated in groups of 10.
